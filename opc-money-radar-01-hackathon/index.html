04OPC 搞钱雷达 · 黑客松篇

一个产品，打多个比赛

别为每个比赛从零做。

这 5 个比赛的内核都是"一个能跑的 AI agent / 工具"。聪明的做法是手里养一个主产品，按每个比赛的要求换壳——AMD 赛把模型调用换成 AMD Cloud，Gemini XPRIZE 接一个 Google Cloud 服务，Band of Agents 把单 agent 拆成多 agent 协调。同一套核心逻辑，投三四个比赛，命中率自然高。

🚀

这周就能做的三步

① 选一个主赛（还在练 → AMD；有用户有收入 → Gemini XPRIZE）。② 今晚免费注册占位，拿到评审标准倒推还差什么。③ 用现有代码改一版，先交一版能跑的再迭代，别新开项目。

🎯
