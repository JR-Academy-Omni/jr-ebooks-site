# Design Language: claude

**权威 spec**：主 repo `curriculum/CLAUDE_DESIGN_STYLE.md`（基于 Anthropic 官方 brand-guidelines skill）— 本文件只记电子书场景的裁剪。

- **一句话气质**：warm, trustworthy, thoughtfully restrained — 像深度杂志，不像交易终端
- **何时用**：Claude / Cowork / Anthropic 主题的工作坊手册、教程书、活动物料
- **核心**：奶油底 `#faf9f5` + 暖黑 `#141413`（拒绝纯黑纯白）+ 赤陶 `#d97757` 唯一主强调；Poppins 标题 + Lora 衬线正文
- **强调循环**：橙 → 雾蓝 `#6a9bcc` → 鼠尾草 `#788c5d`，一个区域只让一个强调色领衔
- **禁**：紫/亮红/亮黄（克制即品牌）；JR neo 元素混入
- **触点**：Claude 主题书通常是 JR 办的活动 → footer 域名保留 jiangren.com.au，contact-bar 照用（组件会自动吃本包配色）
