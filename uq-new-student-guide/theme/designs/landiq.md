# Design Language: landiq

**权威 spec**：`~/Documents/sites/LandIQ/land-iq/rules/design.md`（style reference: Orens Capital）— 本文件只记电子书场景的裁剪。

- **一句话气质**：Professional / Trustworthy / Clean — 机构地产投资感
- **何时用**：LandIQ 合作物料、地产/投资类报告（通常配 `report` 风格皮肤）
- **核心**：深绿 `#134A32` 做结构与 CTA + 浅沙纸底 + 金 `#C4952A` 只做数据/编号点缀；Playfair 衬线 display + 无衬线正文
- **印刷适配**：页底从 web 的 `#E9EAE5` 提亮到 `#F2F2EE`（大面积沙色打印会发灰）
- **禁**：JR 红进入版面；金色做大面积底
- **触点**：合作方物料的 logo / footer / 联系方式**按合作约定**，不默认放 JR 资产 —— 立项时确认归属
