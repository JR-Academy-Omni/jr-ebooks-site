CHAPTER 01

先把词典对齐

同样叫“选课”，每所学校的词不一样。先把词对齐，不然你会把 degree、program、course、unit、subject 混着用，最后连自己到底选了什么都说不清。

词这所学校里大概怎么理解
program你的学位或课程。先看 program requirements。
course单门课。每门 course 有 units、availability、rules。
major / minor主修和辅修路径。要看是否能放进你的 program。
specialisation有些 program 下更细的专业方向，不能和 major/minor 混着猜。
先别急着问“哪门课简单”。更重要的是：这门课能不能算进你的 degree、有没有前置、什么时候开、退课会不会影响学费和签证。

Handbook / catalogue 先看哪几处

Handbook 不是给学校 staff 看的摆设。你每学期少踩的坑，基本都藏在这里。

先看结构Programs and Courses 可以搜 current academic year and beyond 的 programs、majors、minors、courses。
再看单课Degree Builder 是 ANU 这本的主线：先搭 degree，再看 major/minor/course 能不能放进去。
最后做计划读 course entry 时盯四件事：semester、units、prerequisites、delivery mode。
✓

一个真实动作

现在就打开本校官方入口，先搜自己的 degree/program/course，再搜一门你想读的课。把 requirement、availability、requisites 三个词圈出来。你能解释这三个词，才算开始会选课。

CHAPTER 03

Core、elective、major/minor 怎么拆

选课不是把喜欢的课塞满一学期。先把毕业要求拆开，再决定每学期放什么。

你要拆的东西怎么做
毕业结构先把 program requirements 写下来，再决定 major/minor。
选择空间double degree 同学尤其要看两个 program 的重叠和顺序，不要只按兴趣塞课。
顺序约束ANU 的选择空间看起来大，但每个 program 对 level 和 units 都有要求。
实操模板：开一个表格，四列就够：必修 / 主修或方向 / 可选空间 / 不能忘的日期。每门课先放进其中一列，再去排 timetable。

CHAPTER 04

Prerequisite 和 availability，别开学才看

很多选课事故不是系统坏了，是你没看 prerequisites、prohibitions 或开课学期。到 enrolment 那天才发现，已经晚了。

1

规则 1

Course prerequisites 要倒推。比如二年级想读某门课，第一年就要把前置课安排进去。

2

规则 2

如果 course 不是每学期都开，错过一次可能整条路径顺延半年。

3

规则 3

别把同名相近课当替代课，是否能 substitute 要看学校 approval。

选课前自查：
1. 这门课有没有 prerequisite？
2. 我是否已经满足，还是正在读？
3. 它这个 semester / term 开不开？
4. 有没有 prohibition / incompatible / non-allowed subject？
5. 它能不能算进我的 degree requirements？

Enrolment 真正怎么走

计划写得再漂亮，不等于已经 enrol。下面这几步要按学校系统走，别拿别的学校经验硬套。

接受 offer 后还要 enrol

ANU enrolment 页明确，accepted offer 后仍需要 formally enrol。拿 offer 不是自动入读。

用 Degree Builder 做初稿

先把 program、major/minor 和 course 放进去，看结构是不是成立。

进系统前做一次人工核对

确认 prerequisites、semester、units 和 timetable 都能对上，再提交 enrolment。

Timetable：计划和拿到位置是两件事

很多同学以为“我排了课表”就等于“我拿到课”。不是。planner、preference、class registration、allocation，每个词的状态都不一样。

提醒为什么重要
先看状态ANU Timetabling 提供 course timetable information 和 room availability。
再看冲突课表冲突时，先看是否有 alternate class，再看是否要调整 course 顺序。
最后确认远程/线下/混合 delivery 要逐门看，不要假设同专业都一样。
小动作：课表出来后，不只看周几几点。还要看 campus、online/offline、tutorial/lab 是否必须到场、assessment deadline 会不会扎堆。

Census、withdraw、study load：钱和签证都在这里

选错课还能改，错过日期就麻烦了。census date 不是行政术语，它关系到学费、成绩记录和学生签证。

日期提醒：Overload 是单独流程。想多读一门，不等于系统一定允许。
操作提醒：少读、退课、leave 都可能影响 student visa，要先找学校 advice。

CHAPTER 08

FAQ：最常见的 4 个坑

这些问题每年都会有人问。你提前知道，至少不会在开学第一周边哭边改课。

Q

Degree Builder 显示能放进去，是不是就一定能 enrol？

还不够。Degree Builder 帮你看结构，真正 enrol 还要满足 course-level rules 和 timetable。

A

想一学期多读一门课怎么办？

先看 Overload your enrolment。不要默认自己能 overload，通常要满足条件和审批。

Q

Major 以后能换吗？

通常可以调整，但已经读过的 course 能不能算进去要重新核 program rules。

A

课程冲突可以找老师开特例吗？

别先赌特例。先用 timetable 查替代 class 或调整顺序，真无解再找学校 advice。

最后一句：选课不是一次做完的动作，是每学期都要复核的流程。先看官方规则，再问人；别反过来。
