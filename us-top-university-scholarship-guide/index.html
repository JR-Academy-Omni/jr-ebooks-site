这本书怎么读

先说边界：这本主要讲美国顶尖大学本科的奖学金和助学金规则。美国本科申请里，真正影响家庭预算的词通常不是 scholarship，而是 financial aid，尤其是 need-based aid。

研究生不放进同一张表里。MBA、MS、PhD、LLM、医学、法学，每个学院的钱都不一样。有的靠 fellowship，有的靠 TA/RA，有的项目几乎没钱。把它们和本科 full need 写在一起，读者看完只会更糊涂。

核对日期：2026 年 6 月 26 日。所有学校规则都来自学校官网或学校财务援助官网。本书适合作为方向参考；真正递交前，要打开当年申请页面核对你的入学年份、轮次和材料截止日。

CHAPTER 01

先把四个词讲明白

美国顶尖本科的资助逻辑，和很多家长熟悉的“成绩好拿奖学金”不是一回事。尤其是 Ivy+ 私校，很多学校明确写了：本科资助按家庭支付能力算，不按成绩发 merit 奖。

词说白了你要做什么
Need-based financial aid学校按家庭收入、资产、人数、特殊情况等估算你能付多少，差额由学校补准备家庭收入、税务/收入证明、资产证明、CSS Profile/IDOC 或学校自己的表
Merit scholarship按成绩、活动、领导力、特长等给钱先查学校有没有。Harvard、Yale、MIT、Brown 等本科校内基本不发 merit
Need-blind录取时不看你要不要钱放心申 aid，但还要看学校是否 meet full need
Need-aware录取时可能考虑你需要多少资助需要钱仍要如实申，别为了“好录”假装不需要
Meet full demonstrated need学校用自己的公式算出你需要多少钱，然后覆盖 100%别把“我家想少付”当成 demonstrated need。学校会自己算
最重要的一句：如果一个学校规定“申请时没申 aid，录取后不能补”，那你就不能先假装家里能全付，等进去了再说。Stanford、Penn、Cornell、Northwestern、UChicago、Caltech、Duke 都有类似的申请时点要求。

可以期待的

  • Harvard / Yale / Princeton / MIT 这类学校，国际本科也可能拿到非常高的 need-based aid。
  • UChicago 有面向国际 first-year 的 partial-tuition merit，但官网也提醒不要指望 merit 覆盖全部费用。
  • Northwestern 这类学校虽然 need-aware，但录了并给 aid 后，可以 meet 100% demonstrated need。

别这么理解

  • “我成绩好，所以 Ivy 会给我奖学金。”很多 Ivy 根本不发校内 merit。
  • “Need-blind 就是免费。”不是。它只说录取时不看钱。
  • “公立 Top 更便宜，还能给国际生 need-based aid。”很多公立对国际本科非常有限。

一张表看学校分层

先看大图。不是所有“Top 大学”都按同一套规则来，甚至同一所学校对美国学生和国际学生也可能不一样。

类型学校对中国本科申请者意味着什么
A 档：国际 first-year 也 need-blind + full needHarvard、Yale、Princeton、MIT、Brown（Class of 2029 起）、Dartmouth需要资助就申。录取时原则上不因申 aid 吃亏，录取后按学校公式覆盖 demonstrated need。
B 档：need-aware 或国际 aid 名额有限，但录了可能 full needStanford、Penn、Cornell、Johns Hopkins、Northwestern申请 aid 可能影响录取难度，但如果确实需要钱，必须在申请时就申。录后再补风险很高。
C 档：申请时锁定很强UChicago、Caltech、Duke官网明确强调申请时点。没在 admission application 或 admissions process 里申，后面可能不能补。
D 档：公立 Top 要另算UC Berkeley 作为例子官网写国际学生不符合 federal、state 和 need-based university aid。不能把私校 full need 逻辑套到公立。

怎么用这张表：如果家庭预算够全付，B/C 档也可以不申 aid，录取上少一个变量。但如果家庭确实需要资助，不要为了“显得不缺钱”硬撑。顶尖私校一年总成本很高，四年现金流压力不是靠侥幸解决的。

CHAPTER 03

Need-blind + full need 阵营

这几所是很多中国家庭最容易误解、也最值得认真算账的学校。它们不是“看成绩发钱”，而是看家庭支付能力。

Harvard

Need-blindFull needNo merit
  • 国际学生和美国学生适用同一套 aid 规则。
  • Financial aid application 不影响录取。
  • Harvard 本科资助 100% need-based，不发 merit-based awards。
来源：Harvard College Financial Aid

Yale

Need-blindFull needLoan-free package
  • 本科 aid 按 demonstrated need 发。
  • 官网写明不论 citizenship 或 immigration status，meet 100% demonstrated need。
  • Yale Scholarship 只按 financial need，Yale 不发校内 merit-based scholarships。
来源：Yale Affordability / Types of Aid

Princeton

Need-blindSame policy for internationalsPFAA
  • Princeton admission need-blind，申 aid 不在录取中吃亏。
  • 录取后用 grant aid 覆盖 100% demonstrated need。
  • Princeton 用自己的 PFAA，别默认套 CSS Profile。
来源：Princeton Cost & Aid

MIT

Need-blindFull needNeed-based only
  • MIT 官网写明 need blind and full need。
  • 录取学生只要申请 aid，MIT meets full financial need。
  • Financial aid entirely based on demonstrated need, not merit。
来源：MIT Admissions Cost & Aid Basics

Brown

Class of 2029 起国际 first-year need-blindFull need
  • Brown meets 100% demonstrated financial need。
  • 从 Class of 2029 起，国际 first-year applicants 也纳入 need-blind admission。
  • Brown aid 是 need-based，不发 merit awards。Transfer 规则另看。
来源：Brown Financial Aid

Dartmouth

Need-blindFull need
  • Dartmouth 官网首页列出 Need-Blind Admission。
  • 官网写明 meet 100% demonstrated need for all admitted students。
  • 申请前再打开 international students aid 子页核对材料和入学年份截止日。
来源：Dartmouth Financial Aid

CHAPTER 04

Need-aware / 申请时锁定阵营

这组学校最怕“策略性不申”。有些学校明确写了，国际生申请时不申 aid，之后几年不能再申请。你要不要钱，最好一开始就想清楚。

Stanford

International need-awareCSS Profile / IDOC
  • 国际本科有 need-based aid，但名额有限。
  • 国际申请者 request for aid 会作为录取评估因素之一。
  • 如果 admission application 里说不申请 aid，前两年不能申请 Stanford aid。
来源：Stanford International Students

Penn

Need-aware for most internationalsGrant-based full need
  • 除加拿大和墨西哥外，国际申请者 generally need-aware。
  • 符合资格并录取后，Penn 用 grant-based aid meet 100% demonstrated need。
  • 国际学生必须在申请 admission 时申请 financial aid，才有资格在本科阶段拿 aid。
来源：Penn SRFS International Students

Cornell

Need-awareCSS code 2098
  • 国际录取流程 need-aware。
  • 只要你未来任何时候想拿 Cornell grant/scholarship aid，就要申请 admission 时提交 CSS Profile。
  • 录取后 meet 100% demonstrated need，但 package 可能包含 grant、work-study、合理学生贷款。
来源：Cornell International Aid

Johns Hopkins

International need-awareCSS / IDOC
  • Domestic first-year 和 transfer 是 need-blind；国际申 aid 是 need-aware。
  • 国际申请者必须在 application 里 indicate financial need。
  • 若录取，JHU meets 100% demonstrated need。官网 FAQ 提到约 10% 国际学生获得 financial aid。
来源：Johns Hopkins Tuition & Aid

Northwestern

Need-awareFirst-year onlyLoan/work-free
  • 国际学生 request for aid 和申请金额可能影响录取评估。
  • 只有 first-year applicants eligible；必须第一年申请并收到 aid，之后才可继续申请。
  • 录取给 aid 后，meet 100% demonstrated need，且只用 grants/scholarships，不含贷款和 work-study。
来源：Northwestern International Aid

UChicago

申请时必须申Loan-free full need有 partial merit
  • 国际学生应在申请 admission 时同时申请 financial aid。
  • 如果录取且已申请 funding，UChicago meets 100% demonstrated need with a loan-free package。
  • 没在 admissions process 申请 funding，录取后和四年本科期间都不能再申 international financial aid。
  • 国际 aid 申请者不要提交 CSS Profile 或 FAFSA，使用 UChicago Account 里的 Student Financial Aid Worksheet。
来源：UChicago International Financial Aid

Caltech

First-year windowNo aid for intl transfer
  • 国际学生如需 aid，必须作为 incoming first-year 申请。
  • 如果 first-year 没申请，或申请但第一年不符合资格，之后不能再申请。
  • International transfer 和 3/2 applicants 没有 financial aid。
  • 需要在 admission application 里披露 plan to apply for aid。
来源：Caltech International Students

Duke

Financial aid interest checkboxNo intl transfer aid
  • 如果 international student 当初没有在 admission application 勾选 financial aid interest，Duke 官网写明本科阶段不能申请 financial aid。
  • International transfer students 没有 need-based 或 merit scholarships。
  • ED/RD 的 aid deadline 要按当年 Apply Aid 页面核对。
来源：Duke International Students

CHAPTER 05

公立名校别照搬私校规则

很多家庭会把“名校”当成一个类别：哈佛有 full need，那 Berkeley、UCLA、Michigan 也差不多吧？不是。美国公立大学对州内纳税人、州外美国学生、国际学生，是三套预算逻辑。

例

UC Berkeley 的官方口径

Berkeley Financial Aid & Scholarships 官网写得很直接：国际学生不符合 federal、state 和 need-based university financial aid。官网建议国际学生找 private organizations、本国奖学金、研究生 assistantships 等。

来源：UC Berkeley International Students
选校提醒：如果预算必须靠学校补足，私校 full-need 体系和公立大学不在一个逻辑里。公立 Top 可以申请，但不要把它当成“录了就会自动大额减免”的选项。

CHAPTER 06

材料、截止日和申请动作

美国 financial aid 申请不是最后补一个表。真正麻烦的是家庭财务材料：收入、税、资产、翻译、币种换算、非监护父母材料，有的学校还要 IDOC 上传。

Step 1

确定是否申 aid

先按四年总成本算，不要只看第一年。需要学校补差，就在 admission application 里如实选择。

Step 2

查学校用什么表

大多数用 CSS Profile / IDOC；Princeton 用 PFAA；UChicago international aid 用 Student Financial Aid Worksheet，不交 CSS/FAFSA。

Step 3

准备家庭财务材料

收入证明、纳税材料或本国等效文件、银行流水、资产证明、英文翻译、美元换算。家庭结构复杂的，提前查 noncustodial parent 要求。

Step 4

按轮次卡 deadline

REA/ED/RD/Transfer 的 aid deadline 往往不同。不要等 admission 交完才想起财务材料。

几个官网当前可见的 deadline 例子

学校官网当前页显示要记住的事
StanfordREA Nov 15；RD Feb 15；Transfer Mar 15国际生 aid 名额有限，且申请时声明很关键。
CornellED Nov 1；RD Jan 2；Transfer Mar 15CSS Profile code 2098。国际申请者一般没有 CSS fee waiver。
UChicagoEA/EDI Nov 15；EDII Jan 15；RD Feb 15国际 aid 不交 CSS/FAFSA，用 UChicago Account 表格。
NorthwesternED Dec 1；RD Mar 1CSS code 1565；CSS 不可用或 hardship 可看 ISAFA。
DukeED/RD 分轮次列 aid deadline重点不是日期本身，是 admission application 里的 financial aid interest checkbox。
截止日会按申请季更新。这里列的是官网当前页面可见口径，真正提交前按你的入学年份核对。
清

提交前自查清单

一类是申请表动作：Common App / Coalition / 学校系统里是否选择了申请 financial aid，是否完成学校自己的补充问题。

另一类是财务表：CSS Profile、IDOC、PFAA、UChicago Worksheet 或 ISAFA 是否按学校要求提交。

第三类是证明文件：父母收入、税务或等效收入证明、资产、银行材料、英文翻译、美元换算是否一致。数字前后打架，是 financial aid office 最容易追问的地方。

CHAPTER 07

高频误区和 FAQ

误区 1
“Top 大学奖学金靠成绩。”
很多顶尖私校本科不是这么发钱。Harvard、Yale、MIT、Brown 等都把本科资助写成 need-based，甚至明确没有校内 merit awards。
误区 2
“Need-blind 就是我一定不用花钱。”
Need-blind 只是录取时不看你要不要 aid。真正给多少钱，要看学校按公式算出的 demonstrated need。
误区 3
“我先不申 aid，录了再补。”
Stanford 前两年限制、UChicago 四年限制、Caltech first-year window、Duke checkbox、Northwestern first-year only，都在提醒你：这个想法很危险。
误区 4
“公立 Top 更适合低预算家庭。”
州内学生便宜，不等于国际学生便宜。UC Berkeley 官网明确写国际学生不符合 need-based university aid。
误区 5
“研究生照本科 full need 规则。”
研究生要按学院和项目查。PhD 常见 funding，授课硕士常常少很多，MBA/LLM 又是另一套。
Columbia 怎么处理？本次自动抓取 Columbia undergraduate international aid 细则页时遇到官网访问限制。已核到 Columbia College / Engineering 的总体 aid facts 页面，但没有把国际生申请时点规则写进正文。申请 Columbia 的同学，必须用浏览器打开学校官网 international student financial aid 页面复核。
