🟢 CLF-C02 云从业者（入门级）

AWS 云从业者 · 入门级 Foundational · 现行版本 CLF-C02 全篇只采信 AWS 官方（aws.amazon.com / docs.aws.amazon.com）与 Pearson VUE 官方信息，每个关键数字都带官方来源。培训机构软文、第三方题库站、个人博客一律不采信。 数据核对时点：2026 年 6 月。考费随汇率每年至少调一次，报名前请以官网实时为准。

📌 先说一句：版本与定位

CLF-C02 是现行唯一在考的版本。 它于 2023 年 9 月 19 日上线，取代旧版 CLF-C01（C01 最后可考日为 2023 年 9 月 18 日）。网上若有 CLF-C01 旧题库，或"CLF-C03 新版"的说法——前者已退役、后者无官方依据，都别用。

来源：Coming soon: updates to AWS Certified Cloud Practitioner exam · 产品页

这张证在 AWS 全球认证体系里的位置：四级金字塔（入门 Foundational → 助理 Associate → 专业 Professional → 专项 Specialty）的最底层入口。它是整个体系的起点，不绑定具体岗位。

一、证书定位、适配人群、客观含金量

定位

官方原话：面向能够整体展示 AWS 云知识、不限定具体岗位角色（independent of a specific job role）的人。它考的是云的通识认知，不是动手能力——明确超纲的有：写代码、设计云架构、排障、实施、压测。

来源：CLF-C02 官方考试指南

适配人群

  • 官方推荐经验：最多约 6 个月 AWS 接触经验（up to 6 months of exposure），且非强制。
  • 官方点名适合非技术岗：销售、市场、产品/项目经理，以及任何想建立 AWS 云认知的新人。
  • 零基础也能考——这是四级体系里门槛最低的一张，常作为后续考 Associate 的起点。
来源：CLF-C02 官方考试指南 — Target Candidate

含金量，说实话 ⚠️

AWS 不发布任何官方薪资/就业数字。 市面上"持证涨薪 XX"的数据全是第三方调研，本指南不引用、不编造。能确定的是：CLF-C02 是业内公认的 AWS 入门门槛证，价值在于"证明你懂云的基本盘"，适合作为求职简历的起步项或转云方向的第一步，不是工程岗的能力背书（那要看 Associate 及以上）。

重难点拆解

  • Domain 3（34%，最大头）：核心 AWS 服务定位——计算 / 网络 / 数据库 / 存储四大类服务分别干什么、典型场景怎么选。分值最大，优先吃透。
  • Domain 2（30%）：共担责任模型（Shared Responsibility Model） 是必考核心，外加 IAM、安全最佳实践、合规概念。
  • Domain 1（24%）：云的价值主张、Well-Architected Framework、云经济性。
  • Domain 4（12%，最轻）：计费、定价模型、支持计划。
入门考试以"概念识别"为主，不要求动手。重点是把核心服务的"用途"和共担责任模型背熟。

一、这张证是干什么的，适合谁考

它验证什么

官方原话：这门考试面向担任 solutions architect（解决方案架构师）角色的人，验证你基于 AWS Well-Architected Framework 设计解决方案的能力，核心是在 AWS 上设计分布式系统（distributed systems）。

说白了，它不是考你会不会点控制台开一台机器，而是考你面对一个业务场景，能不能选对服务、把架构搭得又安全、又抗造、又快、又省钱——这正好对应它的四个考试域（安全 / 弹性 / 性能 / 成本，后面细讲）。

来源：AWS Certification 官方考试指南合集

适合谁 / 要多少经验

AWS 产品页给的推荐经验（recommended，不是硬门槛）：

  • 至少 1 年动手设计 AWS 云上方案的经验（hands-on experience designing cloud solutions that use AWS services）
  • 有 1–3 年 IT 经验的人适合备考这张证
  • 完全没有 IT 工作经验的，建议先去考 AWS Certified Cloud Practitioner（云从业者，入门级）打底，再回来考这张

注意：这是"推荐"不是"准入"。AWS 没有强制工作年限门槛，报名不需要审你的简历。但经验确实是省备考时间的关键——零基础硬啃也能过，只是周期会拉长。

关于"含金量"，我得跟你说实话 ⚠️

你最想知道的大概是"考了它在美国/中国/澳洲能涨多少薪、好不好找工作"。

这块没有 AWS 官方数字。 AWS 不发布持证人的薪资统计、就业率、含金量排名——一条都没有。市面上那些"AWS 架构师平均年薪 XX 万"的数字，全部来自第三方招聘平台、培训机构、调研公司，正好是本指南的信源铁律要剔除的东西。我不编、也不借第三方数字来撑场面。

能确定的、不掺水的事实是：

  • SAA 是 AWS 认证体系里报考人数最多、岗位需求里出现频率最高的 Associate 级证书之一，定位是云架构通识能力的"敲门砖"。
  • 它的价值在于官方背书的能力证明：招聘方看到它，知道你懂 Well-Architected 那套设计取舍，不用从零考你。

15 道不计分题是什么意思

65 题里有 15 题是 AWS 拿来"试题"的——它们不算你的分，但你考试时看不出来哪些是。所以策略很简单：每一题都当真题做，别纠结"这题是不是送分/废题"。

720 分别误解

scaled score 是换算分，不是"答对几题"。AWS 用换算分是为了让不同套卷（forms）难度可比——你拿到难一点的卷，及格线背后要求的正确率会相应调整。所以别去算"720 分等于答对多少题"，没有固定换算公式。

补充一个体系内的对照（来自官方政策页）：Foundational 级及格 700，Associate 级（含本考试）720，Professional 和 Specialty 级 750。

三、官方考试大纲：四大域 + 权重

SAA-C03 计分内容分四个域（domain），权重如下（合计 100%）：

域名称权重
Domain 1设计安全架构 Design Secure Architectures30% 🥇
Domain 2设计弹性架构 Design Resilient Architectures26%
Domain 3设计高性能架构 Design High-Performing Architectures24%
Domain 4设计成本优化架构 Design Cost-Optimized Architectures20%
来源：SAA-C03 官方考试指南 · 指南 PDF

重难点拆解

Domain 1 安全（30%）—— 权重最高，先啃这个。 官方下分三个 Task：

  • 1.1 设计对 AWS 资源的安全访问（Design secure access to AWS resources）—— IAM 用户/角色/策略、最小权限、跨账号访问、联合身份。
  • 1.2 设计安全的工作负载与应用（Design secure workloads and applications）—— VPC 网络隔离、安全组/NACL、Security Groups、私有子网。
  • 1.3 确定适当的数据安全控制（Determine appropriate data security controls）—— 加密（KMS）、传输/静态加密、数据分类。
安全占 30%，是四个域里最重的，几乎每道场景题都绕不开 IAM 和加密。这块投入产出比最高，优先级拉满。 来源：Domain 1 官方专页

Domain 2 弹性（26%）—— 考"挂了怎么办"。 高可用（Multi-AZ）、容灾、解耦（SQS/SNS）、自动伸缩、跨区复制、备份恢复（RTO/RPO 取舍）。SAA 大量场景题就是给你一个"要求 99.99% 可用"或"某 AZ 故障也不能停"，让你选架构。

Domain 3 高性能（24%）—— 考"怎么更快/更能扛"。 选对计算/存储/数据库类型、缓存（ElastiCache、CloudFront）、读写分离、合适的存储分层、网络优化。

Domain 4 成本优化（20%）—— 考"同样的事怎么更省"。 按需 vs 预留 vs Spot、存储分层（S3 各级别/生命周期）、合理选型避免过度配置。

备考心态：这是"选最优解"的考试

SAA 的题很多是四个选项都能跑通，但只有一个"最符合要求"。题干会给你约束条件（最省钱 / 最高可用 / 运维最少 / 改动最小），你要按约束挑。读题时先圈出关键约束词，比背服务参数更重要。

出分

考完通常很快在 AWS Certification Account 看到通过/未通过，正式 scaled score 报告随后在账号里出。记住：报的是换算分，不是答对题数。

数字徽章（digital badge）怎么领、怎么给人查

AWS 的认证数字徽章由 Credly 平台提供。

  • 怎么领：你会收到一封 Credly 平台发来的邮件，里面有领取（claim）说明；或者直接登录 AWS Certification Account，找 Digital Badges 标签页领。
  • 怎么给别人验证：把你徽章的直链（direct link）发给对方，就能证明你是 AWS Certified；还能建一个公开 profile，把你拿到的所有 AWS 认证徽章集中展示。
  • 徽章会显示你的认证当前是否有效（active），方便雇主随时核到最新状态。
来源：AWS Certification 数字徽章官方页 说明：徽章宿主在 Credly，公开查验走 Credly 出具的徽章直链/公开 profile；本轮未单独固定某条 credly.com 查验 URL，以你领取后徽章本身生成的直链为准。

🆓 免费（Free）：

  • Enhanced Exam Prep Course（增强版备考课） —— 约 6–8 小时，含视频、动手实验（hands-on labs）、更多练习题、并含官方模拟考。

官方四步备考思路

AWS 的备考路径就一句话能概括：自定进度学 → 动手做 → 自测 → 上官方模拟考。对应到资源：

  1. 学：Exam Prep Course（免费 2h）过框架 → 啃官方考试指南 + 各服务的 AWS 文档。
  2. 动手：Cloud Quest / Builder Labs（订阅）实操，架构题靠手感不靠背。
  3. 自测：免费 Official Practice Question Set（20 题）校准。
  4. 就绪测：临考前上 Official Practice Exam（订阅）打全真模拟。

🇨🇳 附录：AWS 全球认证 vs 中国区差异

补充附录，配合本系列各证书指南阅读。 本系列正文讲的全是 AWS 全球认证（Pearson VUE 国际版）。这篇说清楚：考的是全球体系，但你用的 AWS 服务环境可能是中国区——两者不是一回事。 数据核对时点：2026 年 6 月。涉及 AWS 中国区的运营细节以 AWS 中国区官网实际显示为准。

一、先分清两个"AWS"

AWS 全球（Global）AWS 中国区（China）
运营方Amazon Web Services, Inc.北京区域：光环新网（Sinnet）；宁夏区域：西云数据（NWCD）（持牌本地运营商）
账号全球账号独立账号，与全球账号不互通
控制台console.aws.amazon.com单独的中国区控制台域名
计费USD 等国际货币人民币，由本地运营商开票
备案无需面向公网的网站/应用需 ICP 备案
服务范围全部服务、最新特性最先上线部分服务未上线或延后，少数特性不可用
⚠️ 上表中国区运营细节为行业通行事实，具体服务可用性与价格以 AWS 中国区官网 实际显示为准——本附录不逐项编造服务清单。

二、关键认知：考试是"全球统一"的

这是最容易误解的点：AWS 认证考试本身是全球统一的，不分"中国区版"。

  • 你报的认证（CLF / SAA / DVA…）走的是 Pearson VUE 全球考试系统，考纲基于全球 AWS 服务。
  • 也就是说，考试里出现的服务和特性，可能在 AWS 中国区还没上线。如果你日常只用中国区，备考时要特别注意补齐"全球有、中国区暂无"的那部分服务知识。
  • 证书是全球通用的同一张，不存在"只在中国认的 AWS 证"。数字徽章统一走 Credly 查验。
结论：用中国区做项目的同学，备考不能只盯着自己手上的控制台——以官方考纲（docs.aws.amazon.com）的全球服务范围为准。

三、大陆考生报考注意事项

1. 语言

所有现行考试均支持简体中文应考（入门级 CLF-C02 还支持繁体中文）。报名时在 Pearson VUE 选 exam language 即可。用简体中文考则无需申请 ESL 加时。

2. 币种与支付

  • AWS 官方定价表含人民币 CNY：入门 ¥704 / 助理 ¥1,057 / 专业·专项 ¥2,113（每年至少 5 月按汇率更新，可能含增值税 VAT）。
  • 支付支持 Visa / American Express / Mastercard / JCB，以及 voucher（代金券）。
  • 实际扣费币种与金额以 Pearson VUE 报名结算页为准。
  • 来源：AWS 考前政策与定价页

3. 两种考试方式

线下考点OnVUE 居家在线监考
适合家里环境/网络不稳想省通勤、环境可控
中文监考—中国大陆中文（Mandarin）监考时段：周一至五 8:00–17:00（英文监考 24/7）
注意就近选 Pearson VUE 授权考点禁用 VPN/代理、禁虚拟机、需独立安静房间 + 摄像头 + 360° 房扫 + 政府证件
⚠️ OnVUE 明确禁止 VPN/代理，企业网络也常因防火墙不通过。大陆考生选 OnVUE 务必提前跑系统测试、确认网络稳定（家庭宽带通常优于公司网络）；不确定就选线下考点更稳妥。 来源：Pearson VUE OnVUE · 考试中政策

4. 通用规则（与全球一致）

  • 改期：考前 24 小时前，最多 2 次｜取消：考前至少 24 小时否则考费作废
  • 重考：未通过等满 14 个自然日，次数无上限，每次全额付费，通过后同门 2 年内不可重考
  • 有效期 3 年，续证靠重考现行版（专业级可顺带续较低级别角色认证；专项级只能重考现行版续）
